## 功能規劃

### 功能 1：開始頁面
- **輸入**：無（使用者點擊「開始測驗」）
- **處理**：顯示工具標題、簡介、開始按鈕
- **輸出**：進入第 1 題
- **邊界條件**：無

### 功能 2：答題流程（8 題）
- **輸入**：使用者在每題選擇一個選項（4 選 1）
- **處理**：每個選項對應特定角色的加權分數（每個選項可能加分給 1-3 個角色）
- **輸出**：
  - 顯示當前題號 / 總題數（如 3/8）
  - 顯示進度條
  - 選擇後自動滑入下一題（動畫過渡）
- **邊界條件**：不可跳題；可回到上一題修改答案

### 功能 3：結果頁面
- **輸入**：所有 8 題作答完成
- **處理**：累計各角色分數，取最高分角色（若平手取陣列中索引較小者）
- **輸出**：
  - 角色名稱 + 測驗原型標題
  - 角色形象圖（200×200px）
  - 性格描述（3-5 行）
  - 「分享結果」按鈕（複製文字 + 連結）
  - 「重新測驗」按鈕
- **邊界條件**：分享連結應附帶 UTM 參數

### 功能 4：社群分享
- **輸入**：使用者點擊「分享結果」
- **處理**：生成分享文字（含角色名 + 工具連結 + UTM）
- **輸出**：複製到剪貼簿 + toast 提示「已複製！」

## 技術備註

- 計分邏輯：每題 4 選項，每選項可對 1~3 個角色加分（1~3 分），8 題完成後取最高分角色
- 動畫過渡：選擇答案後使用 CSS transition 滑入下一題
- 無後端依賴：所有邏輯前端完成
- 深色主題：預設開啟，使用 UIUX skill 的 CSS tokens
- 零 emoji：所有 UI 圖示使用 AI 生成圖片，禁止使用原生 emoji
